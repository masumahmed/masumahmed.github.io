Sketch the Conductor Website

Description

Music portfolio website for client.

Features

  • Minimalistic, flat and responsive design
  • Use of Instagram API to fetch user posts and display along an infinite scrolling carousel
  • Hands off design, the website automatically updates when a new post is made on Soundcloud or Instagram
  • Languages and frameworks used: HTML, CSS, SASS, JavaScript, jQuery and Bootstrap
  • Use of modular/abstracted web elements for organizational and maintenance purposes
