FAT32 File System

In this project I implemented a functional FAT32 file system using a text file in place of an actual storage device. It demonstrates understading of high level computer science topics in operating systems and file systems.


Example MLK.txt

The system is able to store any 20Kb text file within it's volume. In the example screenshot I used MLK's famous I had a dream speech. In the FAT (left) you can see the file being allocated. In the volume (right) you can see a subsection of the file's content being stored using the index allocation method for file management.

Repo Link: github/masumahmed/filesystem

Link Locker

A website that encodes and decodes URLs with a given password. This application works by using a two way hash algorithm to encode and decode the link. This is useful for poeple that want to use share links with an added layer of security. Like an unlisted Youtube video or ARG game.

Video: https://youtu.be/SxkksvAsUzM

Repo: github.com/masumahmed/link-locker
