Protoflio Website

Description

Personal portfolio website for Masum Ahmed. Used to showcase projects and detail process.

Features

  • Minimalistic, flat and responsive design
  • Use of CSS variables to quickly change the theme of the site (click logo on top)
  • Use of cookie to remember css preferences
  • Modular design - requiring only one file to update to make changes all across the site. Example, navbar menu can be updated all across the site from only using one file
  • Languages and frameworks used: HTML, CSS, SASS, JavaScript, jQuery and Bootstrap
  • Use of abstracted web elements for organizational and maintenance purposes
